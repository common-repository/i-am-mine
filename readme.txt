== Description ==

This isn't an ordinary plug-in.
When this plug-in is done effectively, Makoto Inagaki's profile is shown to the upper right of a management panel besides the plug-in management screen at random.


== Installation ==

1. Upload the entire `caxias-hosoya` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
